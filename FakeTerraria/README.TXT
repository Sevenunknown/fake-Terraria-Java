USER INSTRUCTIONS:
Just run the runner programs main class, and from there you can move left to right with the arrow keys
Jump using the space key, and select to place or break using p and b.
You can also select to place by pressing 1, 2 or 3 showing what slot you'd like to place a block from.
You can can then press the slot you have selected again to deselect placing. Pressing b lets you break any block you'd like.
Enjoy!
